# Accessibility Revisit

**Date:** 2026-03-15
**Status:** Backlog

---

## Context

New features since the initial a11y audit (`docs/reviews/2026-03-15-livery-system-a11y.md`) have introduced additional accessibility concerns.

## Known Issues

- **Flag alt text**: SVG flags use `aria-hidden="true"` on the `<svg>` with `role="img"` and `aria-label` on the wrapper div. Verify screen reader behavior — the `country` field in team data exists for this purpose.
- **Legend grid layout**: Racing Bulls 2-column grid may read oddly in screen readers (visual order vs DOM order). Test with VoiceOver.
- ~~**Chrome legend swatch**: The gradient `"chrome"` swatch has no text fallback for the hex value — screen readers only get the color name "Chrome".~~ Resolved: legend hex values flattened to plain hex codes.
- **Pre-existing issues**: `isWhitePrimary` unused var, contrast ratios, font sizes — see original audit.

## Acceptance Criteria

- All flags are announced correctly by screen readers
- Legend grid reads in logical order
- No new WCAG AA violations introduced since the audit
