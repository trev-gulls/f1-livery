# F1 2026 Livery Design System

Color direction cards for all 11 teams on the 2026 F1 grid.

## What It Renders

A dark-themed card grid where each team gets:

- Custom SVG color swatch with proportional livery representation
- Color legend with hex values
- Finish type (gloss, matte, satin, anodized, metallic)
- Design direction notes and what was removed from prior liveries

Teams: Ferrari, McLaren, Red Bull, Mercedes, Racing Bulls, Aston Martin, Alpine, Duracell Racing, Haas, Audi, Cadillac.

## Tech Stack

- React 19
- Vite 8
- Bun

## Scripts

```sh
bun run dev       # Start dev server
bun run build     # Production build
bun run lint      # ESLint
bun run preview   # Preview production build
```

## Setup

```sh
bun install
bun run dev
```
